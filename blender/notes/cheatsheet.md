**Common used Commands**

	_H_ --> Hide Selectd Objects
	_ALT + H_ --> unhide Selectd Objects
	NUMPADS --> Different Views (Front View)
	Tab  --> Toggle Object / Edit Mode
	G --> Move 
	S --> Scale 
	R --> Rotate 
	G, S, or R + X, Y, or Z --> Action Along Axis
	SHIFT + D  --> Duplicate Object
	Vertex Mode --> 1
	Edge Mode --> 2
	Face Mode --> 3
